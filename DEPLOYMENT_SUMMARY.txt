ğŸ†˜ COMMON ISSUES
--------------------------------------------------------------------------------

Issue: "Command not found"
Solution: npm install -g @railway/cli or npm install -g vercel

Issue: "Database connection failed"
Solution: Check DATABASE_URL includes ?sslmode=require

Issue: "CORS error"
Solution: Update FRONTEND_URL in backend, must match exactly

Issue: "API key invalid"
Solution: Verify Gemini API key is correct, no extra spaces

Issue: "Build failed"
Solution: Check Node.js version (18+), clear npm cache
